🖼️ AI Generative Gallery - v1

A modern web application for managing and displaying AI-generated images with advanced features for metadata extraction and NSFW content management.

Developed by AIVirtus - Majdi El-Jazmawi

✨ Features

Gallery Interface

  • Responsive grid layout
  • Image preview with metadata
  • Lazy loading for performance

Metadata Management

  • Automatic extraction
  • Multiple AI model support
  • Backup system

Content Management

  • NSFW content filtering
  • Blur effects
  • Content warnings

⚙️ Requirements

  • Python 3.8+
  • Flask
  • TensorFlow
  • Pillow
  • SQLAlchemy
  • Better Profanity

🚀 Installation

1. Clone the Repository

git clone https://github.com/yourusername/AIGG.git
cd AIGG

2. Create Virtual Environment

python -m venv venv

3. Activate Virtual Environment

# Windows
venv\Scripts\activate

# Linux/Mac
source venv/bin/activate

4. Install Dependencies

pip install -r requirements.txt

📁 Directory Structure

AIGG/
├── app.py              # Main Flask application
├── requirements.txt    # Python dependencies
├── templates/         
│   └── index.html     # Main gallery template
├── uploads/           # Image storage directory
└── metadata.json      # Image metadata storage

⚡ Usage

  1. Start the application: 
    python app.py
  2. Open your browser and navigate to http://localhost:5000

🗄️ Data Storage

  • Images: Stored in uploads/ directory
  • Metadata: Stored in metadata.json
    • Automatic backup as metadata.json.bak
    • JSON format for easy editing
Note: Make sure to regularly backup your metadata.json file and uploads directory.
